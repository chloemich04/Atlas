# Atlas – Project Plan

## Goal
Build Atlas, a universal tracking platform to develop broad software engineering skills across multiple languages and frameworks.

## Phase 1 – Python Backend (Current Phase)
Stack:
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Pytest
- Docker

## Core Features
- User authentication (JWT)
- CRUD for items
- Tags/categories
- Search and filtering
- Pagination
- Input validation
- Error handling

## Engineering Practices
- Clean project structure
- Clear separation of concerns
- Tests for business logic
- Environment-based configuration
- Git-based workflow

## Future Phases (Do Not Implement Yet)
- Phase 2: Java Spring Boot implementation
- Phase 3: React + TypeScript frontend
- Phase 4: CI/CD, deployment, and polish

## Rules
- Focus on learning, not shortcuts
- Prefer clarity over cleverness
- Avoid overengineering
